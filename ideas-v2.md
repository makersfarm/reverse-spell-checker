# 맏춤법 검사기 v2 — 친근하고 재밌는 디자인

## 선택: 낙서장 + 형광펜 교정지 스타일

**Design Movement**: 손으로 쓴 노트 + 형광펜 교정 + 귀여운 이모티콘 에너지
**Core Principles**:
- 크림/아이보리 배경 (종이 느낌)
- 손글씨 느낌의 타이틀 폰트 (Black Han Sans)
- 형광 노란/핑크/초록 하이라이트로 오류 표시
- 둥글둥글한 카드, 넉넉한 여백

**Color Philosophy**:
- 배경: 따뜻한 크림 #FFFBF0
- 강조: 형광 노랑 #FFE234, 형광 핑크 #FF6B9D, 형광 초록 #7BF5A0
- 텍스트: 진한 차콜 #2D2D2D
- 카드: 흰색 + 살짝 그림자

**Typography**:
- 타이틀: Black Han Sans (굵고 개성 있는 한글)
- 본문: Noto Sans KR 400/500

**Signature Elements**:
- 물결 모양 구분선
- 형광펜 밑줄 효과 (오류 단어)
- 검사하기 버튼에 작은 흔들림 애니메이션
- 오류 뱃지에 이모지
