# 맞춤법 규칙 정리

이 문서는 현재 `맏춤법 검사기`에 들어간 리버스 규칙을 정리한다. 앱은 올바른 표현을 입력받아 사람들이 자주 헷갈리는 표기로 바꾼다.

## 운영 원칙

- 문장부호, 과한 띄어쓰기 장난처럼 사용자를 불쾌하게 만들 수 있는 오류는 넣지 않는다.
- 공식 규정이나 국립국어원 답변으로 확인 가능한 표현을 우선한다.
- 블로그, 커뮤니티, 책 목차, 기사에서 반복적으로 보이는 인기 케이스는 공식 규범과 충돌하지 않을 때만 넣는다.
- 문맥 없이는 위험한 표현은 `contextRequired`로 두고 기본 검사에서는 비활성화한다.

## 되/돼 규칙

기준: `돼`는 `되어`가 줄어든 말이다. `되어`로 풀 수 있으면 `돼` 계열이 가능하고, 풀 수 없으면 `되-`에 다른 어미가 붙은 형태로 본다.

현재 적용:

| 올바른 표현 | 리버스 표현 | 메모 |
| --- | --- | --- |
| 돼 | 되 | `되어`의 준말 |
| 돼요 | 되요 | `되어요`의 준말 |
| 돼서 | 되서 | `되어서`의 준말 |
| 돼야 | 되야 | `되어야`의 준말 |
| 돼도 | 되도 | `되어도`의 준말 |
| 돼라 | 되라 | `되어라`의 준말 |
| 돼야지 | 되야지 | `되어야지`의 준말 |
| 됐다, 됐- | 됬다, 됬- | `되었다`의 준말 |
| 되어 | 돼어 | `되어` 자체가 맞는 표기 |
| 되었- | 돼었- | `되었-` 자체가 맞는 표기 |
| 되나요 | 돼나요 | `되어나요`가 아님 |
| 되나 | 돼나 | `되어나`가 아님 |
| 되는 | 돼는 | `되어는`이 아님 |
| 된, 될 | 됀, 됄 | 관형사형 어미 |
| 된다고, 된다면, 된다 | 됀다고, 됀다면, 됀다 | `되-` 활용 |
| 됩니다 | 됍니다 | `되-` 활용 |
| 되고, 되지, 되면, 되니 | 돼고, 돼지, 돼면, 돼니 | `되-` 활용 |
| 되는데, 되므로, 되면서, 되니까 | 돼는데, 돼므로, 돼면서, 돼니까 | `되-` 활용 |
| 되던, 되더라, 되길, 되기 | 돼던, 돼더라, 돼길, 돼기 | `되-` 활용 |
| 되려고, 되려면 | 돼려고, 돼려면 | `되-` 활용 |
| 됨 | 됌 | `되-` + 명사형 어미 `-ㅁ` |
