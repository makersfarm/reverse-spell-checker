# 맏춤법 검사기 - 디자인 아이디어

## 방향 A: 관공서 패러디 (Bureaucratic Parody)
- **Design Movement**: 90년대 공공기관 웹사이트 + 현대적 재해석
- **Core Principles**: 진지한 척하는 유머, 공신력 있어 보이는 가짜 권위, 레트로 UI 요소
- **Color Philosophy**: 짙은 네이비 + 흰색 + 빨간 강조색 (공공기관 느낌), 진지함 속의 아이러니
- **Layout Paradigm**: 좌측 입력 / 우측 결과 2단 분할 (기존 맞춤법 검사기 레이아웃 패러디)
- **Signature Elements**: 도장 스탬프 효과, "검사 완료" 인증 마크, 오류 개수 뱃지
- **Interaction Philosophy**: 버튼 클릭 시 "딸깍" 느낌의 진지한 로딩 → 황당한 결과
- **Animation**: 결과 등장 시 스탬프 찍히는 효과, 오류 텍스트 하이라이트 순차 등장
- **Typography**: Noto Serif KR (진지한 본문) + 고딕 계열 (UI 레이블)
- **Probability**: 0.07

## 방향 B: 신문사 교열부 (Editorial Desk)
- **Design Movement**: 활자 인쇄소 + 뉴스페이퍼 레이아웃
- **Core Principles**: 활자 질감, 컬럼 구조, 교열 마크업 시각화
- **Color Philosophy**: 크림색 배경 + 먹색 텍스트 + 빨간 교열 펜 색상 (신문 교열 느낌)
- **Layout Paradigm**: 신문 컬럼 그리드, 상단 헤드라인 + 하단 2단 편집
- **Signature Elements**: 빨간 밑줄/취소선 교열 마크, 활자 노이즈 텍스처
- **Interaction Philosophy**: 결과물이 교열된 원고처럼 빨간 펜으로 수정된 느낌
- **Animation**: 빨간 밑줄이 드로잉되는 애니메이션, 교열 마크 순차 등장
- **Typography**: Noto Serif KR + 고딕 계열 혼용
- **Probability**: 0.08

## 방향 C: 클린 모던 (Clean Modern with Playful Twist)
- **Design Movement**: 현대적 SaaS 툴 + 유머러스한 포인트
- **Core Principles**: 깔끔한 2단 레이아웃, 명확한 오류 하이라이트, 재미있는 카피
- **Color Philosophy**: 밝은 흰색 배경 + 짙은 차콜 텍스트 + 오렌지/레드 강조 (오류 표시)
- **Layout Paradigm**: 좌우 2패널 (입력/결과), 상단 타이틀 바
- **Signature Elements**: 오류 하이라이트 뱃지, 오류 개수 카운터, 복사 버튼
- **Interaction Philosophy**: 직관적이고 빠른 피드백, 결과에서 변경된 부분 명확히 표시
- **Animation**: 결과 패널 슬라이드인, 오류 항목 순차 페이드인
- **Typography**: Pretendard + 고딕 계열
- **Probability**: 0.06
