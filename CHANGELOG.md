# Changelog

## v1.0.0 - 2026-05-16

첫 공개 릴리즈입니다.

### Product

- `맏춤법 검사기` 메인 화면을 제공합니다.
- 되/돼, 안/않, 왜/외, 며칠/몇일 등 자주 헷갈리는 한국어 표현을 문장 안에서 표시합니다.
- 맞춤법, 띄어쓰기, 표준어, 외래어 분류를 지원합니다.
- 결과 복사, 공유, 이미지 저장을 지원합니다.
- 빠진 표현을 남길 수 있는 팝업을 제공합니다. 현재 GA4에는 원문 대신 분류와 길이만 기록합니다.

### Operations

- Vercel Git 연동 배포를 설정했습니다.
- Google Search Console, Naver Search Advisor 검증 파일과 SEO 메타를 추가했습니다.
- `robots.txt`와 `sitemap.xml`을 정리했습니다.
- GA4 기본 이벤트와 룰 매칭 이벤트를 추가했습니다.

### Documentation

- README, AGENTS, CLAUDE, 맞춤법 룰 문서를 정리했습니다.
- 룰 업데이트 기준과 현재 적용된 규칙 목록을 문서화했습니다.
